# Project Memory

## Core
ChapterWave — AI eBook creation platform. Dark-mode-first. Primary #4F46E5, Accent #7C3AED, Success #10B981, BG #0F0F1A.
Inter for UI, Lora for book content. Framer Motion animations. Glassmorphism cards.
Lovable Cloud backend. Supabase Auth (email + Google OAuth). Gemini 3 Flash for AI.
Phase 1 MVP: Landing, Auth, Dashboard, Book Wizard, Editor, Profile, PDF+MD export.
